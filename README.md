# chinese-nangong28-guide

本仓库用于归档与发布多个独立的 HTML 页面。这些页面不针对任何特定域名或网站，仅作为通用展示和测试用途。

## 项目简介

chinese-nangong28-guide 是一组静态 HTML 文件的集合。每个文件都是独立的页面，可用于快速浏览、学习参考或作为开发调试的素材。仓库内容会不定期更新，以收录更多不同主题的页面。

## 目录说明

- `pages/` — 存放所有归档的 HTML 页面文件
- `assets/` — 页面引用的资源文件（如图片、样式、脚本等）
- `index.html` — 仓库首页，列出所有归档页面的链接

## 页面归档说明

每个 HTML 页面均为独立完整文件，可直接在浏览器中打开。页面内容可能涵盖常见布局、交互示例、信息展示等类型，不包含任何特定商业或推广信息。

归档的页面仅供学习、测试或参考使用。使用者可根据需要任意下载、修改或部署这些文件。

## 维护说明

本仓库由维护者不定期更新。新增页面会按照时间顺序添加到 `pages/` 目录，并更新 `index.html` 中的链接列表。

如果发现页面无法正常显示或存在明显错误，欢迎提交 Issue 或 Pull Request。请注意，本仓库不接受包含特定域名、网址或诱导点击内容的页面提交。

## 使用方式

1. 克隆或下载本仓库
2. 打开 `index.html` 查看所有页面列表
3. 点击任意页面链接即可直接查看
